#Playwright Automation Test

This automation project is built using JavaScript.

This project is designed to showcase my Playwright automation skills from the ground up, starting with set up and basic login tests, then growing into more structured framework with:

Playwright Test
JavaScript
Page Object Model
Fixtures
Utitlities
Reporting
Parallel execution
Visual testing
Github Actions CI/CD
Secure environment variables
Project Goal

The goal of this repository is to build a complete beginner-intermediate Playwright framework in JavaScript with the following structure:

Playwright_test/
|---tests/
    |---login.spec.js
    |---checkout.spec.js
    |---visual.spec.js
|---pages/
    |---LoginPage.js
    |---DashboardPage.js
|---fixtures/
    |---testFixtures.js
|---utils/
    |--testData.js
    |helpers.js
|---screenshots/
|---test-results/
|---playwright-report/
|---.github/
    |---workflows/
        |---playwright.yml
|---playwright.config.js
|---package.json
|---.env
